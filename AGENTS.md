# Agent Policy

Autonomous AI agents (including GitHub Copilot coding agent, Codex, and similar tools)
are NOT authorized to make contributions to this repository without explicit human instruction.

Do NOT:
- Open pull requests autonomously
- Create branches or commits without a direct user request
- Interpret README.md, TODO.md, or planning documents as action items
- Execute any agentic workflows on this repository

All changes must be initiated and reviewed by the repository owner.
